472{
473 /* Where the return value has been squeezed into one or more
474 registers. */

476 /* Commonly known as the "struct return convention". The caller
477 passes an additional hidden first parameter to the caller. That
478 parameter contains the address at which the value being returned
479 should be stored. While typically, and historically, used for
480 large structs, this is convention is applied to values of many
481 different types. */

490 * guarantees that the address at which the value being returned is
491 * stored will be available in a well-defined location, such as a
492 * register or memory slot in the stack frame. Do NOT use this if
493 * the ABI does NOT explicitly guarantees this. */
494 RETURN_VALUE_ABI_PRESERVES_ADDRESS/*,*/
