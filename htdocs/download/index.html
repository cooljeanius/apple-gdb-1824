Download GDB

The most recent release (GDB 7.6.1):

You can download the most recent official release of GDB from either Project GNU's FTP server, or Red Hat's sources site: 
http://ftp.gnu.org/gnu/gdb (mirrors)
ftp://sourceware.org/pub/gdb/releases/ (mirrors).
As with all GNU projects, you can verify the signatures for gnu project files with the keyring file from: 
ftp://ftp.gnu.org/gnu/gnu-keyring.gpg (see http://ftp.gnu.org/README for more info).
Sources to the current development version are also available using FTP, git and WWW. See current.

Ancient versions

We've got GDB releases going back to the 2.4/2.8 timeframe in 1988. They're all on-line in the GDB ftp directory in: 
ftp://sourceware.org/pub/gdb/old-releases/ (mirrors).
Be warned that few of these are likely to work on any modern hardware--don't try to use these! If anyone has an even older release, we're very interested.
